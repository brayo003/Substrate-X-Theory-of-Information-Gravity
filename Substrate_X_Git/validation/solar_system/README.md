# Solar System Tests

This directory contains validation tests for the Solar System regime.

## Test Cases

1. **Planetary Orbits**
   - Mercury's precession
   - Earth-Moon system
   - Outer planet dynamics

2. **Light Bending**
   - Solar deflection of starlight
   - Shapiro time delay

3. **Pioneer Anomaly**
   - Long-range spacecraft tracking

## Running Tests

```bash
python test_planetary_orbits.py
python test_light_bending.py
```
