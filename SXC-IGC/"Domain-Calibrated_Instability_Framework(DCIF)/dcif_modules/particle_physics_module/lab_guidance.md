# Experimental Guidance for X17 Confirmation

## 1. Detector Requirements
- **Angular Resolution:** < 2.0 degrees (to distinguish X17 bump from IPC background).
- **Energy Resolution:** < 0.5 MeV at 17 MeV.
- **Counting Rate:** Minimum 10^5 pairs to achieve 5-sigma significance for BR ~ 10^-6.

## 2. Material Selection
- **Target:** Neutron-rich targets (e.g., Li-7 for 8Be, B-11 for 12C).
- **Shielding:** Active veto for cosmic muons; lead-shielding for external Gamma-background.

## 3. Falsification Criteria
- If **Neon-20 (17.15 MeV)** shows no peak at **~164 degrees**, the universal vector coupling of X17 is falsified.
- If **NA64** excludes coupling down to **10^-5**, the electronic coupling is falsified.
